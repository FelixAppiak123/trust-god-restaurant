# 🍽️ Food Station Web App

A modern, mobile-first restaurant web application built to handle real-time orders, menu management, and customer interaction without relying on third-party delivery platforms.

---

## 🚀 Features

### 👤 Customer Side

* 📱 Mobile-friendly design
* 🍽️ Live menu (updated in real-time)
* 🛒 Add to cart system
* 📍 Order tracking (live status updates)
* 💬 WhatsApp ordering integration
* 📧 Email-ready structure for confirmations

---

### 🧑‍🍳 Admin Dashboard

* 🔐 Secure login system
* ➕ Add / manage food items
* 📦 View incoming orders
* 🔄 Update order status:

  * New → Cooking → Out for Delivery → Completed
* ⚡ Real-time updates (no refresh needed)

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Responsive Design)
* JavaScript (Vanilla JS)
* Firebase Firestore (Database)
* GitHub Pages (Hosting)

## ⚙️ How It Works

1. Admin adds food items via dashboard
2. Menu updates automatically on the website
3. Customer places order or uses WhatsApp
4. Orders appear instantly in admin panel
5. Admin updates order status in real-time
6. Customer sees live updates
